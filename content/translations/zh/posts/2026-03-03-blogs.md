# 算法笔记

# 链表

- 查询：$O(n)$；增删：$O(1)$
- 特性：长度不固定，可以动态增删，适合频繁增删、较少查询的场景。
- 涉及增删改时，使用虚拟头节点通常更方便；如果只查询，是否使用虚拟头节点差别不大。
- 代码：

```python
class ListNode:
    def __init__(self, val, next=None):
        self.val = val
        self.next = next
```

```cpp
// 单链表
struct ListNode {
    int val;  // 节点上存储的元素
    ListNode *next;  // 指向下一个节点的指针
    ListNode(int x) : val(x), next(NULL) {}  // 节点的构造函数
};
```

# 哈希表

- 快速判断某个元素是否存在于一个“集合”中。
    - 哈希函数：$h(value) = key$，**多对一（key 唯一对应 value）**。
- 查询：O(1)，添加：O(1)，删除：O(1)（平均复杂度）。
- 哈希冲突：链地址法。
    - 需要权衡 *dataSize* 与 *hashSize*。
- 在 Java 中遍历条目：

    ```python
    for (Map.Entry<String, Integer> entry : map.entrySet()) {
    	String key = entry.getKey();
    	int value = entry.getValue();
    }
    ```

- `StringBuilder` 很适合拼接字符串或字符：

    ```java
    StringBuilder sb = new StringBuilder();
    ```

# 双指针

### 总结

- **用两个指针在一个 `for` 循环中完成两个循环的工作。**
- **许多数组或字符串填充问题，可以先把数组扩容到填充后的大小，再从后向前操作。**
- **使用快慢指针：`fast` 每次移动两个节点，`slow` 每次移动一个节点；如果二者途中相遇，则链表存在环。**
- N 数之和：**让前后两个指针不断向中间逼近，在一个循环中完成两层循环的工作。**

# 栈与队列

- 栈：后进先出（LIFO）
    - Java 类：

    ```java
    Deque<String> stack = new ArrayDeque<>();
    stack.push();
    stack.pop();
    stack.peek();
    ```

- 队列：先进先出（FIFO）
    - Java 类：

    ```java
    Queue<String> queue = new ArrayDeque<>();
    queue.offer();
    queue.poll();
    queue.front();

    Deque<String> deque = new ArrayDeque<>();
    deque.offerFirst();
    deque.offerLast();
    deque.pollFirst();
    deque.pollLast();
    deque.peekFront();
    deque.peekLast();
    ```

- 单调队列。
- 优先队列：最小堆或最大堆。

# 二叉树

- **满二叉树**：共有 $2^k-1$ 个节点，$k$ 为树高。
- **完全二叉树**：满二叉树的前 n 个节点，也可以理解为**由若干满二叉树组成**。
- **二叉搜索树**：左子树 < 节点 < 右子树。
- **平衡二叉搜索树**：左右子树高度差不超过 1。
- 遍历方式：
    - DFS：前序、中序、后序（以根节点的位置命名）。
    - BFS：层序遍历。

    ```java
    public void traverseBFS(TreeNode root) {
        Queue<TreeNode> que = new ArrayDeque<>();
        if (root != null) { que.offer(root); }
        while (!que.isEmpty()) {
            TreeNode cur = que.poll();
            if (cur.left != null) { que.offer(cur.left); }
            if (cur.right != null) { que.offer(cur.right); }
        }
    }
    ```

- 要会手动构造一棵二叉树。

```java
public class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;

    TreeNode() {}
    TreeNode(int val) { this.val = val; }
    TreeNode(int val, TreeNode left, TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }
}
```

# 回溯算法

- 回溯 → 递归（用递归树理解）。
- 穷举 → 时间性能通常较差。
- 类型：组合、子集、切割、排列。
    - 同层或路径去重。
    - 预排序后去重。
    - 直接使用 Map 去重。
- **模板**（三步：函数定义、终止条件、单层搜索）：
    - 递归函数通常返回 `void`，结构为 `if + for + 递归 + 回溯`。
    - 递归与 `for`：
        - 递归：纵向遍历。
        - `for` 循环：横向遍历。
    - 伪代码：

    ```cpp
    void backtracking(参数) {
        if (终止条件) {
            存放结果;
            return;
        }

        for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
            处理节点;
            backtracking(路径，选择列表); // 递归
            回溯，撤销处理结果
        }
    }
    ```
