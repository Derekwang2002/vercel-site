PROBABILISTIC INDEX

Skip List 跳表

用多层有序链表模拟平衡索引：底层保存全部元素，上层随机抽样形成稀疏快速通道。

O(log n) 搜索、插入、删除期望复杂度。
p^k 层高按几何概率递减。
update[] 插入删除只改局部前驱指针。

CACHE EVICTION POLICY

LRU Cache

用 HashMap 负责 O(1) 定位，用双向链表负责 O(1) 调整新旧顺序，容量满时淘汰最冷节点。

get O(1) 命中后移动到队头。
put O(1) 更新或插入后刷新热度。
concurrency 比较整表锁、分片和近似 LRU。
